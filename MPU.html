Merchandise Pickup Mobile Application


Sears has a great feature that allowed customers to order items and pick them up in stor with a limited wait time called Merchandise Pick-Up (MPU). It had been a stand-alone application and was going to be integrated into the Shop Your Way suite of apps resulting in a UI makeover.

MY ROLE

I led the re-design and implimentation of additional features such as the timer across Android and iOS devices. The re-design not only included aligning color but I also had the opportunity to optimize many design patterns and the experience as a whole.

MY ROLE

Challenge

On top of aligning the brand palette to match the Shop Your Way suite, we were tasked with incorporating additional features such as the timer which required an entirely new set of user stories to be created in parallel while still taking into account current user flows.

Opportunities

I realized this "reset" of the applications UI style would allow us to take inventory of everything and allow us to develop a style guide which we could then use to port over other application as the suite expanded.

Planning and Scope

I managed to allocate additional time in order to address the increased effort to put together a style guide which would decrease future project timelines and aligned with the UX designers in order to address iOS and Android seperately which had not been done before.
